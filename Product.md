We are building PostShotClarity, a collection of tools aimed at streamers and content creators for repurposing their content.

Each tool is going to be independent, standalone, and covers a specific use case. We are going to have a directory of tools that is easily searchable, and the UI of everything should look very, very cohesive.

We want a sidebar, with the design:

PostShotClarity (header)

(Tabs)
Emotes Formatter
GIF Generator from Clips
Horizontal to Vertical
Clip Searching
Reel Editing 

(bottom aligned buttons)
Support
Theme

For the emotes formatter, we want it so that the user can upload an image and get all the different sizings for it back, so that it is easily uploadable as emotes for platforms like Twitch.
For the GIF Generator, the user should be able to:
Upload a clip between X to X seconds
Pick the right second duration for the limits
Convert the uploaded video clip into a GIF.
For the horizontal to vertical tool, we want it so that the user can upload a horizontal video clip and position the vertical 9x16 ratio over the horizontal clip to decide which part of the horizontal clip they want to keep as the vertical clip. The user should also be able to grab the right spot of where their webcam is in the original video, and be able to place their webcam over the right spot in the vertical video. We want this to be useful for repurposing things like gaming clips into short form vertical reels for TikTok/ Instagram etc.
